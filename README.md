# Supplementary Materials for Robust Video Watermarking

This repository provides the supplementary experimental video materials for the manuscript:

**Robust Video Watermarking Algorithm Based on Compression-Domain Motion Gating**

The full video files are stored in Baidu Netdisk because of their large file sizes. This repository provides the access link, file structure, and the mapping relationship between original videos, watermarked videos, watermark messages, platform-return videos and the corresponding experimental result files.

## Baidu Netdisk Link

Baidu Netdisk link: https://pan.baidu.com/s/1WyPNDS5UVWE2hSAEswUDog?pwd=gsdn

Extraction code: gsdn

## Contents

The supplementary materials include four types of files:

1. Original test videos.
2. Watermarked videos generated by the proposed method.
3. Embedded watermark messages.
4. Videos downloaded from three real video platforms after uploading the watermarked videos.
5. The decoding results for each sample from the real platform experiment and the platform-level statistical results.

The three tested video platforms are:

- Bilibili
- Douyin
- Kuaishou
