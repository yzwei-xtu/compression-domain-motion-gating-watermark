# compression-domain-motion-gating-watermark
Experimental supplementary materials for robust video watermarking based on compression-domain motion gating.

# Supplementary Experimental Videos

This repository provides supplementary experimental materials for the paper:

**Robust Video Watermarking Algorithm Based on Compression-Domain Motion Gating**

The supplementary materials include original test videos, watermark messages, watermarked videos, and decoding results.
